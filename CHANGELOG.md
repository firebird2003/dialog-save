# Changelog

## [v2.6.0] - 2026-03-20

### 新增功能

#### 1. 硬链接支持
- 新增 `--use-hardlink` 配置选项
- 支持硬链接替代软链接，解决 Obsidian 无法识别软链接的问题
- 硬链接文件在 Obsidian 中可直接编辑和查看

#### 2. 多代理支持完善
- 修复 `get_all_agent_sessions()` 扫描逻辑
- 支持自动发现所有代理目录
- 代理配置文件路径查找优化

#### 3. 监控服务增强
- 添加心跳检测机制
- 自动重启功能
- 详细的错误日志

#### 4. 命令行改进
- 新增 `--verbose` 详细输出模式
- 新增 `--dry-run` 预览模式
- 新增 `--force` 强制执行选项

### 问题修复

1. **版本号统一**：统一所有文件的版本号为 v2.6.0
2. **目录结构**：统一使用 `claw-对话/` 目录结构
3. **配置文件**：修复配置文件路径处理问题
4. **错误处理**：增强错误处理和日志记录

### 技术改进

- 代码模块化：拆分 manage.sh 为多个功能模块
- 性能优化：减少不必要的文件系统操作
- 兼容性：支持 macOS 和 Linux

## [v2.3.0] - 2026-03-17
